添加图片也有两种方法，就像链接一样，并且渲染出来的结果是一样的。 添加链接和图片的区别仅仅在于前面的感叹号 ( ! )。

6.引用块

如果您需要为其他来源的内容添加特殊的格式来引起读者的注意， 或为其他杂志的文章设计特殊的格式，那么 Markdown 的_引用块_语法将会非常有用。 引用块是一个拥有特殊格式的句子或段落，为了引起读者的注意。例如：

“无为而治的罪是所有七种罪中最致命的。有传言说，要使邪恶的人实现其目的，只需要好人无所事事。”

要创建一个引用块，您只需要在行首添加大于号（>）。例如：

“无为而治的罪是所有七种罪中最致命的。有传言说，要使邪恶的人实现其目的，只需要好人无所事事。”

做的不错！

您可以在每一行引用前都添加一个大于号。 当你有多个段落时，这显得很有用。例如：

他的话似乎打动了他自己的天性。他是否说过自己，说过自己是或希望成为自己？
斯蒂芬沉默地看着自己的脸。那里充满了悲伤。他谈到自己，担心自己的孤独。

“你在和谁说话？” 史蒂芬问道。

克兰利没有回答。

注意：即使是空行也必须包含大于号。这可以确保引用是一个整体。

引用块内可以使用其他的 Markdown 格式，例如斜体，图片或超链接。

Markdown 是一种轻量级标记语言，常用于编写文档、博客、邮件等等。下面是 Markdown 语法的一些具体内容：

  1. 标题：使用 # 来表示标题，# 的数量表示所属的级别，例如 # 一级标题，## 二级标题，以此类推。
  2. 段落：使用空行来分隔开不同的段落，Markdown 会自动将它们转换成不同的段落。
  3. 列表：使用 * 或者 - 开始一行，表示无序列表；使用数字加 . 开始一行，表示有序列表。
  4. 引用：使用 > 开始一行表示引用，可以多层嵌套，例如 >> 意味着嵌套的引用。
  5. 粗体和斜体：使用 ** 文字 ** 来表示粗体，使用 * 文字 * 表示斜体。
  6. 链接：使用 链接文字 来表示链接，例如 Markdown 教程。
  7. 图片：使用 来表示图片，例如 。
  8. 代码：使用 `代码内容` 来表示行内代码，使用 ``` 代码块内容 ``` 来表示多行代码块。
  9. 分割线：使用三个或以上的 - 或者 * 来表示分割线，例如 —- 或者 ****。
  10. 表格：使用 | 来分隔不同的列，使用 - 来表示表头与表内容的分隔线，例如：

以上是 Markdown 语法的一些基本内容，还有其他的语法标记可以根据需要进一步了解。因为 Markdown 语法易读易写，所以被广泛应用于各种场合。
